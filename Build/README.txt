-------- COLLECTOR.EXE -----------
Phần mềm sử dụng để lấy và gộp thông tin 9 profile từ 1 profile nào đó.

Sử dụng lệnh trên cmd: collector.exe <Path to folder containing profile folders> <Profile Num>

Sau khi chạy sẽ xuất ra file tương ứng với profile trong thư mục DATA.

-------- DISTRIBUTOR.EXE ---------

Phần mềm thêm dữ liệu lấy từ collector vào 1 profile nào đó.

Sử dụng lệnh trên cmd: distributor.exe <Path to folder containing profile folders> <Profile Num>


Dữ liệu token đầu vào được đặt trong thư mục DATA.
